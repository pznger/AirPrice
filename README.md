# AirPrice
使用 Python 的 pandas 、matplotlib、 sklearn 、requests 、scipy 、flask 等库，在 github 公开的数据集上以及爬取得到的 天气数据整合为合并数据集，通过随机森林方法预测统一航线的不同班次票价情 况，并构建一个可视化与预测查询平台，意在为航空出行提供方便，把握票价波动。
